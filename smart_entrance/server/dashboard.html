⚡ Quick Setup Guide

1

Install & Run Server

Install dependencies and start this Flask server on your PC.

pip install -r requirements.txt
python server.py
2

Start Ngrok Tunnel

Expose your local server to the internet so ESP32-CAM can reach it.

ngrok http 5000

Copy the https://xxxx.ngrok.io URL

3

Flash ESP32-CAM

Upload the esp32_cam.ino sketch. Paste your Ngrok URL + WiFi credentials into the config section.

serverURL = "https://xxxx.ngrok.io/detect-face"
